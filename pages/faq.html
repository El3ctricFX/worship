FAQ

Frequently asked questions & keyboard shortcuts

How can I contribute to the website or request an addition?

You can do this through the website's GitHub Repository. If you know how to code yourself, you can submit a pull request. If you just want a feature added, submit through GitHub's Issues tab as a Feature Request & Enhancement!

What keyboard shortcuts are available?

Here are all the shortcuts built into the website:

  • 1, 2, 3 – Navigate to Bible Verses (1), Testimonies (2), Resources (3)
  • 1 (from any inner page) – Return to homepage
  • ESC – Return to homepage
  • ? or f – Open this FAQ page

Also, clicking on the header text (e.g., "Bible Verses" while on that page) will take you back home.
